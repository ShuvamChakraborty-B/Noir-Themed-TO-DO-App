# Noir-Themed-TO-DO-App
A film-noir–inspired To-Do list application designed to look like an old detective case board. The application allows users to add, update, complete, filter, and delete tasks while storing all data locally through LocalStorage for persistent sessions. Built entirely with vanilla HTML, CSS, and JavaScript, it features category filtering (All, Active, Completed), a clear-completed workflow, and dynamic UI updates based on user actions. The interface blends minimal design with a noir aesthetic, demonstrating practical DOM manipulation, state management, event-driven logic, and clean code structure without any external libraries or frameworks.
